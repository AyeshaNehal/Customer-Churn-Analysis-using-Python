#  Customer Churn Analysis using Python

##  Project Overview

This project presents an end-to-end customer churn analysis using Python. The objective is to identify the factors that contribute to customer churn by analyzing customer demographics, subscription details, billing information, and customer support interactions.

The project follows a complete data analytics workflow, including data cleaning, preprocessing, feature engineering, exploratory data analysis (EDA), visualization, SQL database integration, and business insight generation.

---

##  Objectives

* Clean and preprocess customer data.
* Merge multiple datasets into a single analytical dataset.
* Perform exploratory data analysis (EDA).
* Engineer meaningful business features.
* Visualize customer churn trends.
* Generate actionable business insights and recommendations.

---

##  Dataset

The project combines three datasets:

* **Customer Dataset** – Customer demographics and personal information.
* **Subscription Dataset** – Subscription plans, billing details, and churn information.
* **Support Dataset** – Customer complaints, escalations, and customer satisfaction (CSAT).

### Final Dataset

* **23 Records**
* **21 Features**

##  Key Insights

* Approximately **34.8%** of customers in the dataset have churned.
* Customers subscribed to the **Basic Plan** exhibit the highest churn rate.
* **Referral** customers have the highest churn compared to other acquisition channels.
* Customer complaints and escalations can indicate a higher likelihood of churn.
* Churn risk segmentation helps identify customers who require proactive retention efforts.
* Correlation analysis reveals relationships between customer attributes and churn-related variables.

---

##  Business Recommendations

* Improve the value proposition of the Basic subscription plan.
* Enhance onboarding and engagement strategies for referral customers.
* Monitor high-risk customers using churn scores.
* Improve complaint resolution time and customer support quality.
* Develop targeted retention campaigns for customers with high churn probability.
* Continuously monitor churn metrics through dashboards.
